Troca de Peças

Arraste as peças com o mouse para inverter as posições das peças pretas e brancas.

Faça uma jogada válida para começar.

Regras

  • Deslize uma peça para a casa vizinha se ela estiver vazia.
  • Pule exatamente uma peça para a casa seguinte, se ela estiver vazia.
  • Objetivo: deixar as peças brancas à esquerda e as pretas à direita.
